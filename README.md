## 📋 What is rdt-cli?

rdt-cli lets you browse Reddit directly in your Windows terminal or command prompt. You can read feeds, open posts, search for topics, upvote posts, save items, and subscribe to subreddits, all without opening a web browser.

This app works with simple commands typed into the terminal window. It’s built for users who want fast and distraction-free access to Reddit.

---

## 🖥️ System Requirements

- Windows 10 or later
- A terminal or command prompt program (built-in to Windows)
- An internet connection

No special tools or installations are needed besides the steps below.

## 💡 Basic Commands

Here are some commands you can use inside rdt-cli:

- `feed` — View your Reddit home feed  
- `read <post number>` — Open a post to read its comments  
- `search <keyword>` — Find posts using keywords  
- `upvote <post number>` — Upvote a post  
- `save <post number>` — Save a post for later  
- `subscribe <subreddit>` — Follow a new subreddit  
- `help` — See a list of all available commands  
- `exit` — Quit the app

Use arrow keys or page keys to scroll through long lists.

## 🔧 Adjusting Settings

You can customize rdt-cli with simple configuration files stored on your computer.

By default, the app creates a folder named `rdt-cli` inside your Documents folder. Inside, you will find a `config.txt` file.

Open `config.txt` in a text editor like Notepad to change:

- The number of posts displayed per page  
- Whether posts open in light or dark terminal colors  
- Default subreddit for feeds  

Save the file and restart rdt-cli for changes to take effect.

## 🛠️ Troubleshooting

- **App doesn’t start:**  
  Make sure you installed the `.exe` file correctly. Try running the installer again.  
- **Terminal shows “command not found”:**  
  Restart your computer to refresh system paths. Confirm rdt-cli installed correctly.  
- **No posts loading:**  
  Check your internet connection. Reddit must be accessible.  
- **Text looks strange or unreadable:**  
  Adjust your terminal font or window size.

If problems persist, look for the latest software updates on the release page.
